textpipe.data.cleaner module#

textpipe.data.cleaner.clean_text(text)[source]#

Clean the input text by removing punctuation (except apostrophes), converting to lowercase, and normalizing whitespace.

Parameters:

text (str) – Input raw text

Returns:

Cleaned text with consistent formatting

Return type:

str

Example

>>> clean_text("Hello, world! How's it going?")
'hello world how's it going'
>>> clean_text("   Space   and   punctuation!   ")
'space and punctuation'
textpipe.data.cleaner.remove_stopwords(text, stopwords)[source]#

Remove stopwords from text while preserving original word order.

Parameters:

  • text (str) – Input text to process

  • stopwords (set) – Set of stopwords to remove

Returns:

Text with stopwords removed

Return type:

str

Example

>>> stopwords = {"is", "the", "in", "a", "to"}
>>> remove_stopwords("This is a sample text", stopwords)
'sample text'

textpipe.data.tokenizer.basic_tokenizer(text)[source]#

Tokenizes text by: - Lowercasing - Splitting contractions via apostrophes - Preserving @ and # symbols - Splitting on spaces

Parameters:

text (str) – Input text to tokenize

Returns:

List of tokens

Return type:

list

Example

>>> basic_tokenizer("Python's great, isn't it?")
['python', 's', 'great', 'isn', 't', 'it']
